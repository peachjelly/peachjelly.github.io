C 언어의 정의
-------------
* C언어<br>
  프로그래밍 언어로 고급 언어지만 하드웨어 제어가 가능(포인터를 이용한 메모리 접근)하여 저급 언어의 특성을 지님
  * 저급 언어(high-level)<br>
  인간이 이해하기 쉬운 언어
  * 고급 언어(low_level)<br>
  컴퓨터가 이해하기 쉬운 언어 (어셈블리어)
* 기계어<br>
  0, 1로 구성된 컴퓨터가 사용하는 언어
* 컴파일러<br>
  프로그래밍 언어로 작성된 프로그램을 컴퓨터가 이해할 수 있도록 기계어로 번역해주는 번역기<br>
  소스 프로그램을 한번에 기계어로 변환시킴
  * 어셈블러(assembler)<br>
    기호로 표현된 어셈블리 코드를 기계어로 번역하는 번역기
  * 인터프리터(interpreter)<br>
    프로그램을 한 단계씩 기계어로 해석해여 실행하는 언어처리 프로그램
* 사람 - 컴파일러 - 컴퓨터
  해당 순서로 상호 소통
  
C 언어의 특징
-------------
* 프로그램 이식성이 높음
* 저급 언어의 특성을 가진 고급 언어

C 프로그램의 완성 과정
-------------
1. 코딩 단계<br>
  소스코드를 작성하여 소스파일을 생성하는 과정
2. 컴파일 단계<br>
  소스파일이 목적 파일로 변환되는 과정
    * 목적 파일(object file)
      * .obj라는 확장자를 갖는 파일
      * 컴퓨터가 이해할 수 있는 기계어들의 집합으로 변경된 상태
      * 목적 파일은 그 자체로는 큰 의미를 갖지 못하고 라이브러리라는 파일과 연결되어 하나의 프로그램이 되었을 때에만 의미를 가짐
3. 링킹 단계<br>
  목적 파일을 실행 파일로 변환하는 과정으로 컴파일된 목적 파일들을 라이브러리 파일과 연결하여 실행 파일을 만들어 냄
